# Form Validation App

A simple form validation project built using Vanilla JavaScript. This application focuses on validating user input on the client side without using any backend or data storage.

## Features

* Validate required fields
* Check email format
* Validate password rules (length, format, etc.)
* Show error messages in real-time
* Prevent form submission if inputs are invalid

## Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)

## Live Demo

View the project here:

## How It Works

The form listens for user input and submit events. JavaScript handles validation logic by checking each field against predefined rules. If any input is invalid, error messages are displayed and the form submission is blocked.

## Purpose

This project was built to practice:

* DOM manipulation
* Event handling
* Input validation logic

## Limitations

* No data is saved
* No backend integration
* Validation only works on the client side
